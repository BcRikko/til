[text-wrap: balance](https://developer.mozilla.org/ja/docs/Web/CSS/text-wrap#balance)を使うことで、`<wbr>`を使うことなく日本語をいい感じに折り返し処理してくれる。

```css
.text-wrap {
  text-wrap: balance;
}
```

## Before

```text
Lorem ipsum dolor sit
amet, consectetur adipiscing
elit 
```

最後の行だけ微妙…


## After

```text
Lorem ipsum dolor
sit amet, consectetur
adipiscing elit
```

その点、`text-wrap:balance`はすげぇよな。最後まで文字たっぷりだもん。
