[TypeScript 5.3](https://devblogs.microsoft.com/typescript/announcing-typescript-5-3-beta/)で最新のImport Attributes（`with`）をサポートした。

[Import AttributesのProposal](https://github.com/tc39/proposal-import-attributes)で紆余曲折あり、`with { type: ... }`に決まりそうだ。


## 変遷
- 2020-06: `import … from … with { type: 'json' }` / Stage 2
- 2020-09: `import … from … assert { type: 'json' }` / Stage 3
- 2021-11: TypeScript 4.5で`assert { type: … }`をサポート
- 2023-01: Stage 2 に戻る
- 2023-03: `import … from … with { type: 'json' }` / Stage 3
- 2023-10: TypeScript 5.3 で`with { type: … }`をサポート
