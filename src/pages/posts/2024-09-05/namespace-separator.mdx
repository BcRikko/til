CSSで見かけないセレクタ（`|`）があった。

「|」は[Namespace Separator（名前空間セパレータ）](https://developer.mozilla.org/en-US/docs/Web/CSS/Namespace_separator)と呼ばれるもので、セレクタと[名前空間（@namespace）](https://developer.mozilla.org/ja/docs/Web/CSS/@namespace)を分離するために使われる。

```css
/* XHTMLの名前空間*/
@namespace url(http://www.w3.org/1999/xhtml);
/* SVGの名前空間 */
@namespace svg url(http://www.w3.org/2000/svg);

a {
  color: red;
  text-decoration: underline dashed;
}
svg|a {
  fill: green;
  text-decoration: underline solid;
}
```
