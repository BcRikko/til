[place-items](https://developer.mozilla.org/ja/docs/Web/CSS/place-items)プロパティを使うとグリッドやフレックスボックスなどにおいて、[align-items](https://developer.mozilla.org/ja/docs/Web/CSS/align-items)と[justify-items](https://developer.mozilla.org/ja/docs/Web/CSS/justify-items)を1行で指定できる。

```html
<section id="demo">
  <div class="item"></div>
</section>
```
```css
.section {
  display: grid;
  place-items: center;
}
```


## デモ

<section id="demo">
  <div class="item"></div>
</section>

<style>{`
#demo {
  display: grid;
  width: 100%;
  height: 200px;
  background-color: #f0f0f0;
  border: solid 4px;
  place-items: center;

  .item {
    width: 50px;
    height: 50px;
    background-color: #333;
  }
}
`}</style>
