日本語では`「」`や`【】`などの括弧や`、`や`。`などの句読点を使うことが多い。ただ、それらは全角扱いなので先頭が揃わない、文中に余分なスペースが生まれるなどの弊害があった。

[text-spacing-trim](https://developer.mozilla.org/en-US/docs/Web/CSS/text-spacing-trim)を使うことで、先頭や文中の括弧や句読点のスペースを調整することができる。

## ## ブラウザ対応状況

現在Experimentalとして一部のブラウザで実装されている。

- Chrome 123
- Edge 123

## text-spacing-trim

- `normal`: 基本括弧句読点は全角で行に収まらないときは調整される
- `space-all`: すべての括弧・句読点は全角になる
- `space-first`: normalと同じだが明示的な改行のあとには全角になる
- `trim-start`: 各行の先頭のい半角になる以外はnormalと同じ


```css
p {
  text-spacing-trim: normal;
  text-spacing-trim: space-all;
  text-spacing-trim: space-first;
  text-spacing-trim: trim-start;
}
```
