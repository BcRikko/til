## テスト対象のHTML
```html
<a>リンクです</a>

<nav class="breadcrumb">...</nav>
```

## テスト
```css
a:not([href]) {
  --error: 'hrefがありません。もしかしてbutton?';
}

.creadcrumb:not(:is(nav[aria-label], nav[aria-labelledby])) {
  --error: 'パンくずリストにラベルがありません';
}
```
