Sassの`.Block { &__Element: ... }`のように`&`でCSSカスタムプロパティを拡張する[CSS Variable Group](https://github.com/w3c/csswg-drafts/issues/9992)という提案がなされている。

便利そうだけど、`&__Selector`と同じようにGrep検索でヒットしなくなるので、ご利用は計画的にって感じ。

## Before

```css
:root {
  --color-green-010: ...;
  ...
  --color-green-100: ...;
}

color: var(--color-green-100);
```

## After
```css
:root {
  /* CSS Variable Groupで`--color-green`というprefixに対し、010〜100のsufixをつける */
  --color-green: {
    010: ...;
    ...
    100: ...;
  }
}

color: var(--color-green-100);
```
