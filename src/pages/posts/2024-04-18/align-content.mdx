## align-content を使った上下中央揃え

flexやgrid用のコンテナを用意しなくても[align-content](https://developer.mozilla.org/ja/docs/Web/CSS/align-content)だけで上下中央揃えができるようになった。

```css
.container {
  align-content: center;
}
```

## その他の方法

```css
/* flexbox */
.container {
  display: flex;
  align-items: center;
}

/* grid */
.container {
  display: grid;
  place-items: center;
}

/* line-height */
.container {
  height: 100px;
}
.inner {
  line-height: 100px;
}

/* absolute */
...省略...

/* transform */
...省略...
```
