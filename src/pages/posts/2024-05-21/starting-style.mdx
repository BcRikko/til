[@starting-style](https://developer.mozilla.org/en-US/docs/Web/CSS/@starting-style)を使うと、トランジション開始時のスタイルを定義できる。

`@starting-style`はトランジションのみで、CSSアニメーションはいままで通り`@keyframes` + animationを使う。

- ※2024-07-21現在、Chrome117、Edge117、Safari17.5で利用可能。
- FirefoxはExperimentalとして実装されており、flagを有効にすることで使える

```css
.target {
  transition: background-color: 1.5s;
  /* トランジション後のスタイル */
  background-color: green;
}

@starting-style {
  .target {
    /* トランジション前のスタイル（初期表示時のスタイル） */
    background-color: transparent;
  }
}
```
