Firefox121が[:has()](https://developer.mozilla.org/en-US/docs/Web/CSS/:has)をサポートしたことで、全主要ブラウザで利用可能になった。

以前NES.cssを開発しているときに`:has()`が使えず苦労したので、見た目優先のUIライブラリを開発している人にとっては嬉しい機能だ。

## Before
```css
/* 苦肉の策で書いたコード */
.nes-container.with-title {
  ...
  & > .title { ... }
}
```

## After
```css
// :has() が使える時代
.nes-container:has(.title) {
  ...
}
```
