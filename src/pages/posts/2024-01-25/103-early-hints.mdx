MDNを見ていたところ知らん[HTTPレスポンスステータスコード（103 Early Hints）](https://developer.mozilla.org/ja/docs/Web/HTTP/Status/103)が増えていた。


```text
Browser               Server
  |------- GET -------->|
  |                     |
  |<-- 103 Early Hints -|
  |                     |
  |-- GET /image.png--->| 103 の Linkヘッダにかかれているリソースを表示前にロードする
  |                     |
  |<----- 200 OK -------|
  |<--- /image.png -----| <img src="/image.png"> などのリソースは 103 が返ってきた時点で
  |                     | 事前に読み込みにいっているので表示が速くなる

```
上図のようにデータをレスポンスする前に、必要なリソースをあらかじめブラウザに伝えることで表示速度を向上させることができる。

Firefox 120でサポートされたことで、一部制限はあるが全主要ブラウザで利用可能になった。
