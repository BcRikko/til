ESLint実行時に[--cache](https://eslint.org/docs/latest/use/command-line-interface#caching)フラグをつけることで、`.eslintcache`が作成され変更があったファイルのみチェックするようになるので実行時間が短くなる。

```sh
$ npm run eslint -- --cache
```
