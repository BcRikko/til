いままで[Masonryレイアウト（組積レイアウト）](https://developer.mozilla.org/ja/docs/Web/CSS/CSS_grid_layout/Masonry_layout)をするにはJavaScriptが必須（jQueryのプラグインとか）だったが、Gridレイアウトの機能でCSSだけで実現できるようになった。


```css
.masonry {
  display: grid;
  gap: 10px;
  /* grid-template-(rows|columns): masonry が指定可能 */
  grid-template-rows: masonry;
}
```

## 追記:2024-04-24

- [Safari TP](https://webkit.org/blog/15269/help-us-invent-masonry-layouts-for-css-grid-level-3/)がMasonryレイアウトをサポートした。
- Firefoxは`layout.css.grid-template-masonry-value.enabled`を有効にすることで利用可能

----

## 追記:2024-05-08

Chromeの開発チームがCSS Gridを使った[Masonryレイアウトの実装に反対の意を示した](https://developer.chrome.com/blog/masonry)。

### 反対の理由

1. パフォーマンス問題
2. Grid ではなく別のレイアウトにすべき

### Chromeチームの提案

CSS Gridの機能ではなく、`display: masonry`でフローレイアウトの役割のひとつとして実装することを提案している。
```css
.masonry {
  display: masonry;
}
```

----

## 追記: 2024-09-26

[CSS Grid Level 3のWorking Draft](https://www.w3.org/blog/CSS/2024/09/24/css-grid-3-fpwd/)が公開された。

`grid-template-rows: masonry`と`display: masonry`の2つの提案があり、ユーザーにフィードバックを求めている。
