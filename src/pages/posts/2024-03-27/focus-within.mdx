[:focus-within](https://developer.mozilla.org/ja/docs/Web/CSS/:focus-within)を使うと、その要素、または子要素にフォーカスが当たったときにスタイルを当てられる。

```html
<form>
  <div class="field">
    <label for="one">One</label>
    <input id="one">
  </div>
  <div class="field">
    <label for="two">Two</label>
    <input id="two">
  </div>
</form>
```

```css
/* いったん親要素:focus-withinからlabelにたどって変更する*/
.field:focus-within label {
  font-weight: bold;
}

/* 他の方法: すぐとなりにフォーカスがあたったinput要素があるときにlabelのスタイルを変更する  */
label:has(+ input:focus) {
  font-weight: bold;
}
```
