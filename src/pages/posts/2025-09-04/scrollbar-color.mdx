[scrollbar-colorプロパティ](https://developer.mozilla.org/en-US/docs/Web/CSS/scrollbar-color)を使うことで、スクロールバーの色を変更できる。

[Safari TP 227](https://webkit.org/blog/17324/release-notes-for-safari-technology-preview-227/)でサポートされたので、近々全主要ブラウザで利用可能になりそうだ。

<Baseline featureId={"scrollbar-color"} />


```css
.container {
  /* User-Agentのスタイルが適用される*/
  scrollbar-color: auto;

  /* 1つ目:スクロールバーのつまみ、2つ目:スクロールバーのトラック */
  scrollbar-color: #eee #333;
}
```

- auto: デフォルトのスタイル、User-Agentやプラットフォームのスタイルが適用される
- `<color> <color>`: スクロールバーのつまみとトラックの色を指定する
