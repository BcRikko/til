[Firefox127](https://developer.mozilla.org/en-US/docs/Mozilla/Firefox/Releases/127)が[Clipboard API](https://developer.mozilla.org/en-US/docs/Web/API/Clipboard_API)のRead/Writeをサポートしたことで、全モダンブラウザで利用可能になった。


```js
// コピー
const text = 'xxxx'
await window.navigator.clipboard.writeText(text)

// 貼り付け
const paste = await window.navigator.clipboard.readText()
```

※ページにフォーカスが当たっていない場合、`DOMException: Failed to execute 'readText' on 'Clipboard': Document is not focused`というエラーになるため、Consoleからの実行はできない。
