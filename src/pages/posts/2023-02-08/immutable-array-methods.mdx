[EMCAScript proposal](https://ecmascript-daily.github.io/ecmascript/2023/02/03/ecmascript-proposal-update)が更新され、Arrayにイミュータブルメソッドが生えてくるらしい。

## Before

`Array#reverse()`を使うと元の配列の中身まで書き換えてしまう。
```js
const list = [1,2,3,4]
list.reverse()

console.log(list)  // [4,3,2,1]
```

## After

`Array#toReversed()`を使うと元の配列は書き換えられない。

```js
const list = [1,2,3,4]
const reversed = list.toReversed()

console.log(list)      // [1,2,3,4]
console.log(reversed)  // [4,3,2,1]
```
