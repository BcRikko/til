内包するコンテンツが溢れたときにスクロール or 非表示にするときには[overflowプロパティ](https://developer.mozilla.org/en-US/docs/Web/CSS/overflow)を使うのが一般的である。しかし、overflowプロパティは視覚的にX軸またはY軸のどちらかを指定してスクロールを発生させるため、横書きと縦書きコンテンツの両方に対応できない。（対応する場合は2パターン用意する必要がある）

[overflow-block](https://developer.mozilla.org/ja/docs/Web/CSS/overflow-block)や[overflow-inline](https://developer.mozilla.org/en-US/docs/Web/CSS/overflow-inline)は、従来のoverflowプロパティと同様にコンテンツが溢れたときにスクロール or 非表示を設定できる。さらに、論理的プロパティなので`writhing-mode`が変わっても対応できる。

```css
.article {
  overflow-block: auto;
  overflow-inline: hidden;
}

.rl {
  writing-mode: vertical-rl;
}
.tb {
  writing-mode: horizontal-tb;
}
```
