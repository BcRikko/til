[search要素](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/search)が追加されたことで、[ARIAランドマークロール](https://developer.mozilla.org/ja/docs/Web/Accessibility/ARIA/ARIA_Techniques#%E3%83%A9%E3%83%B3%E3%83%89%E3%83%9E%E3%83%BC%E3%82%AF%E3%83%AD%E3%83%BC%E3%83%AB)がすべて揃った。

## Before

```html
<form role="search">
  ...
</form>
```

## After

```html
<search>
  <form>...</form>
</search>
```

## ARIAランドマーク一覧

- banner … `<header>`
- complementary … `<aside>`
- contentinfo … `<footer>`
- form … `<form>`
- main … `<main>`
- navigation … `<nav>`
- region … `<section>`
- search … `<search>`
