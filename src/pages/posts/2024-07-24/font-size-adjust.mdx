[font-size-adjust](https://developer.mozilla.org/en-US/docs/Web/CSS/font-size-adjust)がBaselineに到達した。

`font-size`に同じ値を指定しても、フォントによってはサイズ感が異なることがある。そこで`font-size-adjust`に係数を指定することで、フォントのサイズ感を調整できる。

```html
<p class="verdana">Lorem ipsum dolor sit amet, consectetur adipiscing elit</p>
<p class="times">Lorem ipsum dolor sit amet, consectetur adipiscing elit</p>
<p class="adjust-times">Lorem ipsum dolor sit amet, consectetur adipiscing elit</p>
```

```css
html {
  font-size: 1.25rem;
}

.verdana {
  font-family: Verdana, sans-serif;
}
.times {
  font-family: Times, serif;
}
.adjust-times {
  font-family: Times, serif;
  /* font-size * font-size-adjust でサイズ感を調整 */
  font-size-adjust: 0.5;
}
```
