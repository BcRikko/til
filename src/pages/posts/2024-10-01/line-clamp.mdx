[Safari TP 204](https://webkit.org/blog/15978/release-notes-for-safari-technology-preview-204/)がリリースされ、[line-clamp](https://developer.mozilla.org/ja/docs/Web/CSS/-webkit-line-clamp)プロパティが使えるようになった。

他のブラウザは未対応のため、`-webkit-`ベンダープレフィックス付きのプロパティをいくつか使う必要がある。


```html
<p>Lorem ipsum ...</p>
```

```css
/** Safari TP 204 */
p {
  line-clamp: 3;
}

/** その他のブラウザ */
p {
  display: -webkit-box;
  -webkit-box-orient: vertical;
  -webkit-line-clamp: 3;
  overflow: hidden;
}
```
