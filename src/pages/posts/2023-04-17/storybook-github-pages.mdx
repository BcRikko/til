Storybookで生成したファイルをGitHub Pagesでホストするとエラーになる。

## 原因

- Storybookで生成したファイルに`_commonjsHelpers-xxxxxx.js`のようにアンダースコアで始まるファイルがある
- GitHub PagesのデフォルトのサイトジェネレータであるJekyllはアンダースコア始まりのファイルを無視する
- 結果、必要なファイルが読み込めずにエラーになる

## 解決策

- Rootに`.nojekyll`という空ファイルを作って、Jekyllを無効化する
