Chrome115/Edge115でCSSだけで[スクロール駆動アニメーション](https://developer.mozilla.org/en-US/docs/Web/CSS/animation-timeline)ができるようになった。

※2023-07-25時点で、Firefox、Safariは未対応。

```html
<main>
  スクロールが発生するくらい長いコンテンツ
</main>

<span class="point"></span>
```
```css
@keyframes move {
  from {
    left: 0;
  }
  to {
    left: calc(100vw - 100px);
  }
}

.point {
  ...
  animation: move linear;
  /* scroll(scroller, axis)を指定することで
   * animationプロパティに指定したmove keyframesのアニメーションが実行される
   */
  animation-timeline: scroll(root block);
}
```

## デモ

スクロールすることで、ページ下部にある赤いバーが連動してアニメーションすることが確認できる。

※2023-07-25時点ではChrome115、Edge115でのみ確認できる
