HTMLの`<menu>`要素は廃止されたり、復活したりしている。

Living Standardには[The `menu` element](https://html.spec.whatwg.org/#the-menu-element)があるので使っても問題なさそう。

- HTML 2.0 … 標準化
- HTML 4.01 … 非推奨
- HTML5 … 廃止
- HTML 5.1 … 復活
- HTML 5.2  … 再度廃止
- HTML Living Standard … 復活
