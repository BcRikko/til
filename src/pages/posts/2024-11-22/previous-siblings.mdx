[:has()](https://developer.mozilla.org/ja/docs/Web/CSS/:has)擬似クラスと兄弟セレクタ（次兄弟結合子）を使うことで、兄要素が取得できる。

対応ブラウザは以下の通り。

import Baseline from '../../../components/Baseline.astro'

<Baseline featureId={"has"} />

```html
<span>兄</span>
<span>弟</span>
```

```css
/* 兄要素 */
span:has(+ span)::after {
 content: "より優れた";
}

/* 弟要素 */
span + span {
 content: "は存在しない";
}
```
