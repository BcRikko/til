主にリンクに使われる[text-decoration](https://developer.mozilla.org/ja/docs/Web/CSS/text-decoration)の下線の位置を[text-underline-offset](https://developer.mozilla.org/ja/docs/Web/CSS/text-underline-offset)を使うことで調整できる。

対応ブラウザは以下の通り。

import Baseline from '../../../components/Baseline.astro'

<Baseline featureId={"text-underline-position"} />


```css
.text {
  text-decoration: underline;
  text-underline-offset: 1em; /* <length> */
  text-underline-offset: 30%; /* <percentage> */
  text-underline-offset: -10px; /* negative */
}
```

pxやemなどの`<length>`や`<percentage>`の値を指定できる。また、負の値も指定できる。

  <p class="text" style="text-underline-offset: 30%;">text-underline-offset: 30%; … gjpqy</p>
  <p class="text" style="text-underline-offset: -5px;">text-underline-offset: -1em; … gjpqy</p>
