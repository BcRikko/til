日本語では気になる場面は少ない。しかし、英語の場合は、テキストの上下のスペースにより中心線がズレて見えたり、スペースのリズムが崩れているように見えたりする。

`text-box`プロパティを使うことで、開発者自身がテキストの上下のスペースをカスタマイズできるようになった。

<Baseline featureId={"text-box"} />

## 書き方
```css
text-box-trim: trim-start | trim-end | trim-both | none;
text-box-edge: cap | ex | text | leading;
line-fit-edge: alphabetic | text;

/* ショートハンド */
text-box: trim-both cap alphabetic;
```

## 例

```css
h1 {
  /* 上下を大文字英字にあわせる */
  text-box: trim-both cap alphabetic;

  /* 上下を小文字の「x」に合わせる */
  text-box: trim-both ex alphabetic;
}
```
