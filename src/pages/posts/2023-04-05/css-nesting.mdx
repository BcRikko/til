[Chrome 112](https://developers-jp.googleblog.com/2023/04/chrome-112.html)でついに待望の[CSS Nesting](https://developer.mozilla.org/ja/docs/Web/CSS/CSS_nesting)が使えるようになった。

CSS Nesting、CSS Variables（Custom Properties）などがサポートされることで、Sassの機能の9割（自分調べ）をカバーできるようになる。
```css
.foo {
  .bar {
    &:hover {}
  }
}
```
