コンテンツの増減によりスクロールバーが表示されたり消えたりするときに、レイアウトがガタつくことがある。これを防ぐために[scroll-gutterプロパティ](https://developer.mozilla.org/en-US/docs/Web/CSS/scrollbar-gutter)を使うことができる。

[Safari TP 203](https://webkit.org/blog/15860/release-notes-for-safari-technology-preview-203/)が対応したことで、近い将来全主要ブラウザで利用可能になる。

## サポート状況

- Chrome 94
- Edge 94
- Firefox 97
- Safari TP 203

## scrollbar-gutter

```css
/**
 * デフォルト: スクロールバー分のスペースは確保されない
 */
.container {
  scrollbar-gutter: auto;
}
/**
 * あらかじめスクロールバー分のスペースが確保される
 */
.container {
  scrollbar-gutter: stable;
}
/**
 * 両サイドにスクロールバー分のスペースが確保される
 */
.container {
  scrollbar-gutter: stable both-edges;
}
```
